My Dot Files
============

In this repository you can take a look in my dot files.

Enjoy =)

**1. Install Dependencies**

```bash
sudo apt-get install \
    clang-format     \
    exuberant-ctags  \
    git              \
    tmux             \
    vim
```

**2. Install Dotfiles**


```bash
./install.bsh --all
```
